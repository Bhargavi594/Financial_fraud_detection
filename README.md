💳 AI Financial Fraud Detection System
📖 Description

The AI Financial Fraud Detection System is a machine learning–based project designed to identify suspicious financial transactions and predict the probability of fraud in real time.
The system analyzes transaction patterns such as transaction amount, account balance changes, and transaction type to determine whether a transaction is legitimate or fraudulent.
This project simulates how modern banks and fintech companies detect fraudulent activity automatically.

⚙️ Technologies Used

The system is built using the following technologies:
Python for machine learning and backend logic,
Scikit-learn for training the fraud detection model,
Pandas & NumPy for data preprocessing and analysis,
Flask REST API for real-time fraud prediction,
PostgreSQL for storing transaction records,
OpenCV / Streamlit (optional) for visualization or dashboard

Development was done using
Visual Studio Code and version control with
GitHub.

🚀 Features
With this system, you can:
 Analyze financial transaction datasets,
 Predict fraud probability using machine learning,
 Detect suspicious transactions in real time via API,
 Store and manage transaction records in a database,
 Monitor fraud activity using a dashboard
